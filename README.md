# AI会改变人们用软件的交互方式。不需要那么多UI设计了。
- 人类需要方便的观察界面，只是chat窗口太过于单调。即便是AI会在窗口中给出图片等等易于观察的UI，也比鼠标点一下慢，更何况AI还好反应，还要给指令。AI的每次执行还需要消耗比较多的电力、算力。
  - 所以，起码一部分管理系统是不会被取代的。
  - 但是，photoshop、3D设计软件什么的不好说，设计过程会变化，但是不会被取代。而且这个需要点时间吧。
  - 日本的开发模式会有什么变化呢？
  - CRM，ERP，等管理系统的形态呢？
    - **这个是AI时代的创意过程，是获得AI红利的一种方式** = **AI原生的，对CRM，ERP等等应用生产的系统的形态的重构、设想**
- AI完成实际UI的做成，当时创意、指示是来自于prompt。
- 和应用场景结合的prompt有一定的存在价值。->那还是有prompt工程师的需求啊。
- 回到UI的话题，对话接口是一种方式，视频也是，短视频也是。
- 另外一种是AI对AI的，或者说机器对机器的，这部分应该是高效的通信方式。
- 那么，AI可以创造出新的数据交换方式。
- 那6G、7G是可以由AI来完成设计吗。起码会参与到设计中的。
